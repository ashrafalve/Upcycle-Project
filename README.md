# UpCycle - Sustainable Marketplace

A Next.js marketplace for buying, selling, and upcycling pre-loved items.

## Features

- **Marketplace** - Browse and purchase upcycled & pre-loved items
- **User Roles** - Buyers, Sellers, Designers, and Admin
- **Product Listings** - Create listings with images, pricing, and condition details
- **Upcycle Requests** - Request custom upcycling projects from designers
- **Designer Portfolio** - Showcase upcycling work and accept projects

## User Roles

- **Buyer** - Browse, purchase items, request upcycling
- **Seller** - List items for sale
- **Designer** - Accept upcycling projects, manage portfolio
- **Admin** - Manage users, products, and requests

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Zustand (State Management)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Project Structure

```
app/               # Next.js App Router pages
components/        # Reusable UI components
features/         # Feature-specific components
services/         # API services
store/            # Zustand stores
```
